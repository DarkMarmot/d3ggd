D3 Foundations

To play with D3, there are a couple main concepts to explore.

  • SVG Tags -- like HTML for shapes
    • g is like the div for everything
    • text tags are needed are text -- tagless stuff doesn't work
    • rect (rectangle)
    • circle
    • path is for lines, the 'd' attribute contains instructions (data) used to build the line
    • style things with strokes and fills
  • Selections -- like jQuery, but just different enough to cause massive pain and confusion
  • Method Chaining -- most methods return the selection they act on
  • Functions are used as configurable objects all over the place
  • Enter, Update and Exit
  • Select (does not change bindings) vs SelectAll (creates a data bound selection)
